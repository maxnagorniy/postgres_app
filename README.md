
**This is Fullstack App**
=====================

Stack:
   -
   - express
   - knex
   - postgresql
   - ...
   

   
Start Project
   - 
   
    npm install -g knex 
    npm install
   
   ##### knex migrations
  
   
   Create database postgres 'todos_test'
   
    knex migrate:rollback
 
    knex migrate:latest
    
    knex seed:run
   
    npm start
